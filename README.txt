1)Расчёт эффективных интервалов.
	1.1) Для расчёта эффективных интервалов сначала необходимо запустить ICA: Tools -> Run ICA
	1.2) Расчёт эффективных интервалов: Tools -> Calculate effective intervals
		1.2.1) Аргументы
			а) Left interval from event (ms) - Количество миллисекунд слева от метки (всегда положительное)
			б) Right interval from event (ms) - Количество миллисекунд справа от метки (всегда положительное)
			в) Left constraint from interval start (ms) - Количество миллисекунд справа от интервала, заданого параметрами (а) и (б).
				Т.е. значение 0 соответсвует значению, заданному параметром (а)
			г) Right constraint from interval start (ms) - Количество миллисекунд справа от интервала, заданого параметрами (а) и (б). Правая граница 
			д) Coefficient of superiority (>1) -  Коэффициент превосходства. Определяет во сколько раз должна превосходить негэнтропия на интервале, который не входит в интервал, заданный границами (в) и (г), но входит в интервал между (а) и (б), чтобы данный интервал попал в результат.
			е) Minimal interval length (ms) - Минимальная длина эффективного интервала.
			ж) Select events - Метки, относительно которых будут рассчитаны эффективные интервала.
			
			ВАЖНО: Параметры (в),(г) и (д) устарели. Рекомендуется исопльзовать значение (в) = 0, (г) равному сумме (а) и (б), и (д) - дефолтное значение (1). Эти параметры могут быть использованы для задания границ наиболее вероятного расположения эффективных интервалов.
				Также рекомендуется задавать в параметре (ж) метки отсчёта времени эпох (в 0 мс)
			Пример:	(a): 500, (б): 1500, (в): 0, (г): 2000, (д): 1, (е): 5, (ж): 1 3  

2) Визуализация эффективных интервалов на Component activations: Plot -> Plot effective intervals

3) Расчёт ERSP Вариант 1:
	effint_newtimef(EEG, EFFINTS, comp_num, event_types, cycles, varargin)

	Аргументы:
	EEG - Глобальная переменная EEG
	EFFINTS - Глобальная переменная EFFINTS
	comp_num - номер компоненты
	event_types - Массив строковых названия меток
	cycles - аналогично newtimef
	varargin - аналогично newtimef

	Выход: аналогично newtimef

	В данном варианте ERSP расчитвается ТОЛЬКО среднем значении эффективного интервала для всех эпох, соответсвующих ТОЛЬКО заданным меткам. В результате разложение может произойти на малое число частот, т.к. ЭИ малы для построения.

	Пример:
		[ersp,itc,powbase,times,freqs,erspboot,itcboot] = effint_newtimef(EEG, EFFINTS, 7, ['S  1'; 'S  2'], [0.3 0.5], 'plotersp',  'on')

4) Расчёт ERSP Вариант 2:
	effint_newtimef_full(EEG, EFFINTS, comp_num, event_types, tlimits, cycles, varargin)
	
	Аргументы:
	EEG - Глобальная переменная EEG
	EFFINTS - Глобальная переменная EFFINTS
	comp_num - номер компоненты
	event_types - Массив строковыхназвания меток
	tlimits - аналогично newtimef
	cycles - аналогично newtimef
	varargin - аналогично newtimef

	Выход: Первое значение - ersp на эффективном интервале. Остальные аналогичны newtimef

	В данном варианте ERSP рассчитывается на всём интервале, заданном параметром tlimits, Только на эпохах, соответвующих меткам, заданным параметром event_types. При отображении ERSP и ITC границы эффективных интервалов отображаются на графиках.
	
	Пример:
		[effint_ersp, ersp,itc,powbase,times,freqs,erspboot,itcboot] = effint_newtimef_full(EEG, EFFINTS, 1, ['S  1'; 'S  2'], [-1000 2000], [0.3 0.5], 'plotersp',  'on');
